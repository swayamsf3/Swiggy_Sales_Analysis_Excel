## Swiggy Sales Analysis | Excel

This project focuses on analyzing Swiggy sales data using Microsoft Excel.
The objective is to understand city-wise sales distribution, popular food categories,
and peak order periods through data analysis and dashboarding.

### Tools Used
- Microsoft Excel (Pivot Tables, XLOOKUP, Charts, Map Visualization)

### Project Highlights
- Analyzed sales data to identify top-performing cities and food categories
- Created pivot-based summaries for daily and monthly performance
- Designed an Excel dashboard with charts and map visualization
